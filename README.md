# Muhammad Adil Academic Website

This repository contains the GitHub Pages website for Muhammad Adil.

## Publish

1. Upload all files and folders to the root of `madil5.github.io`.
2. Open **Settings → Pages**.
3. Choose **Deploy from a branch**.
4. Select **main** and **/(root)**.
5. Save.

The website will be available at:

https://madil5.github.io

## Updating content

Edit the relevant `.html` file directly on GitHub or replace the file with an updated version.
